# Amazon Front Page Clone

A professional-grade structural clone of the Amazon homepage. This project focuses on semantic HTML markup and responsive CSS styling to replicate the layout and aesthetic of one of the world's most complex e-commerce interfaces.

## Features
- Semantic HTML: Utilizes modern HTML5 elements for improved accessibility and SEO.
- Responsive Layout: Engineered with Flexbox and CSS Grid to maintain structural integrity across various screen sizes.
- Detailed Styling: Accurately replicates navigation bars, product grids, and footer layouts.

## Technology Stack
- HTML5
- CSS3 (Vanilla)

## Usage
Simply open the index.html file in any modern web browser to view the static clone.
